# Wallet Web

Interface web para o sistema de carteiras digitais.

## Tecnologias
- **React** + **TypeScript**
- **Vite**
- **Tailwind CSS**
- **Axios**
- **React Router DOM**

## Como rodar
```bash
# Instalar dependências
npm install

# Rodar a aplicação
npm run dev
```

## Pré-requisito
O back-end (wallet-api) precisa estar rodando em `http://localhost:3000`
